SessiyaApp - Приложение для планирования задач и сессий, менеджер задач с календарем, напоминаниями и профилем пользователя. Разработано на SwiftUI для демонстрации навыков iOS-разработки.

Особенности:
- Онбординг для студентов
- Календарь с выбором недели/дня
- Создание/редактирование задач (title, description, time)
- Профиль пользователя
- Локальное хранение
- Dark Mode support(будет добавлено)
- SwiftUI + MVVM архитектура

Технологии
- Swift 5.9+
- SwiftUI
- Combine / async-await (если используешь)
- UserDefaults (для хранения)
